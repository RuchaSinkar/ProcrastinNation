# ProcrastinNation 

ProcrastinNation is a simple productivity-focused web application designed to help users track sessions, manage time better, and fight procrastination (one task at a time).

This project was built as a hands-on learning project to understand full-stack development concepts, backend APIs, and real-world project structure.

---

## Features

- Session-based tracking
- Backend server with database integration
- Clean and minimal frontend
- Simple and understandable project structure

---

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

   ```bash

    ```bash
   cd ProcrastinNation
3. Install dependencies
    ```bash

    ```bash
